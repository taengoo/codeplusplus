In Section 1, “Flying Before You Walk,” you put together the framework of your first HTML5 mobile game and got a spaceship flying around the screen. Until this point what's been built so far is more a toy than a game. To make it a game, you need to add some enemies and set up the various elements of the game so that they can interact with each other.
